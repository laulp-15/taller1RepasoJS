Async / Await en JavaScript

¿Qué es Async/Await?

Async/Await es una forma más sencilla y legible de trabajar con promesas. Permite escribir código asíncrono como si fuera síncrono. "Async" indica que función manejará promesas. "Await" espera el resultado de una promesa.
